  - writes `real/lineup.csv`

  - `real/.cache/realsports_multiplier/`

Auth / session notes:

- Real Sports password login was getting rate-limited with:
  - `Attempts exceeded, please press the "Forgot password?" button`
- The working path is browser-backed session reuse, not repeated `/login`
- Active local auth/session files:
  - `real/.realsports_auth_cache.json`
  - `real/.realsports_browser_session.json`
  - `real/.realsports_env.ps1`
- Those files are useful to copy to another desktop, but they may expire or stop working across machines
- If they do expire, reseed auth on the new desktop with:
  - `python real\bootstrap_realsports_session.py`
  - or `python real\extract_realsports_session_from_chrome.py`
- If browser extraction fails, capture a fresh authenticated Real Sports request header from Chrome on that machine and reseed the session from that
